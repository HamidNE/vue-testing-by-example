# 8. Emitted event

```html
<template>
  <div>
    {{ data }}
  </div>
</template>

<script>
  export default {
    data: () => ({ data: 'DATA' }),
    methods: {
      submit() {
        this.$emit('submit', { data: this.data })
      }
    },
  };
</script>
```
